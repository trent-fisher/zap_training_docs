# Contact Us Enquiries Training Guide

## Overview
View all contact form submissions from your website. This is a **view-only** page - to assign agents or manage inquiries, go to the **Leads page**.

Navigate to: **Admin → Enquiries → Contact Form Enquiries**

---

## Table Columns

- **First Name** / **Last Name** - Customer name
- **Email Address** - Contact email
- **Phone No.** - Contact number
- **Subject** - Inquiry topic (truncated, click eye icon for full text)
- **Message** - Inquiry content (truncated, click eye icon for full text)
- **Status** - Unassigned (orange) or Assigned (green)
- **Agent** - Assigned agent name
- **Actions** - Eye icon to view full details

---

## Searching & Filtering

### Global Search (searches everything)
Type to search across: name, email, phone, subject, message, agent, status



### Column Filters

**First Name**: Click filter icon → type name

**Status**: Click filter icon → select Unassigned or Assigned

**Agent**: Click filter icon → select one or more agents

---

## Viewing Full Details

1. Click **eye icon** in Actions column
2. Modal shows:
   - **Assigned Agent** (if assigned)
   - **Update info** (who updated, when)
   - **Name, Email, Phone** (customer contact info)
   - **Subject** (full text)
   - **Message** (full text)

---

## Managing Inquiries

**You cannot assign or update from this page.**

To manage an inquiry:
1. Note customer email or name
2. Go to **Admin → CRM → Leads**
3. Search for the customer
4. Look for lead source: "Contact Us Form"
5. Assign agent and manage there

---

## Export Data

Click **"Download"** button → exports all inquiries to CSV

Includes: all names, contacts, full subjects, full messages, status, agents

---

## Status Meanings

**Unassigned (Orange)**: No agent assigned yet - needs immediate attention

**Assigned (Green)**: Agent assigned and handling

Status updates happen in Leads page, reflect here automatically.
