# Package Enquiries Training Guide

## Overview
View all package enquiry submissions from your website. This is a **view-only** page - to assign agents or manage enquiries, go to the **Leads page**.

Navigate to: **Admin → Enquiries → Package Enquiries**

---

## Table Columns

- **Package Title** - Name of the package (truncated if long)
- **Name** - Customer first and last name
- **Email Address** - Contact email
- **Phone No.** - Contact number
- **Status** - Unassigned (orange) or Assigned (green)
- **Agent** - Assigned agent name
- **Actions** - Eye icon to view full details

---

## Searching & Filtering

### Global Search
Type to search: package title, name, email, phone, agent, status

### Column Filters

**Package Title**: Click filter icon → type package name

**Name**: Click filter icon → type customer name

**Email Address**: Click filter icon → type email

**Phone No.**: Click filter icon → type phone

**Status**: Click filter icon → select Unassigned or Assigned

**Agent**: Click filter icon → select agents (multi-select)

---

## Viewing Enquiry Details

Click **eye icon** → modal shows:

### Header
- **Reference Code**: "Ref: [package_reference_code]"
- **Assigned Agent** (if assigned)
- **Update info** (who updated, when)

### Customer Info
- Name
- Email Address
- Phone Number

### Travel Dates
- **Depart Date**: Departure date
- **Return Date**: Return date
- Visual: Plane icon → dashed line → Location pin

### Package Information
- **Package ID**: Clickable link to view package on website
- **Package Title**: Full package name
- **Message**: Customer's message/inquiry about the package

---

## Managing Enquiries

**You cannot assign or update from this page.**

To manage:
1. Note customer email/name
2. Go to **Admin → CRM → Leads**
3. Search for customer
4. Look for lead source: "Package Enquiry Form"
5. Assign and manage there

---

## Export Data

Click **"Download"** → exports all package enquiries to CSV

---

## Status Meanings

**Unassigned (Orange)**: Needs assignment

**Assigned (Green)**: Agent handling
